## Multiclass Classification using Artificial Neural Networks (ANN)

### 📌 Project Overview

The goal of this project is to demonstrate how a simple Feed-Forward Neural Network can learn patterns from a structured dataset to perform accurate predictions.

    Task: Multiclass Classification

    Model Type: Artificial Neural Network (ANN)

    Library Used: TensorFlow, Pandas, Numpy, & more.

### 📊 Dataset

    Dataset Name: Crop Recommendation Dataset
    Number of Class/Crops: 22

    Source Link: https://www.kaggle.com/datasets/atharvaingle/crop-recommendation-dataset


### 📈 Performance Results

After training with dropout regularization & Adam optimization algorithm, the model achieved the following performance on the test set:

1. Accuracy: 92%
2. Loss: 0.2
